# Portfolio

Personal portfolio website with video background and asterisk mask design.

## Project Structure

```
Portfolio/
├── frontend/           # Frontend React application
│   └── portfolio-app/  # Vite + React 18 app
│       ├── src/        # React components and styles
│       ├── public/     # Static assets (videos, SVG masks)
│       └── ...
├── backend/            # Backend server (Coming soon)
└── README.md
```

## Getting Started

### Frontend

```bash
cd frontend/portfolio-app
npm install
npm run dev
```

The app will be available at `http://localhost:5173/`
